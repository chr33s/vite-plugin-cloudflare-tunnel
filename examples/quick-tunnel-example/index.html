🌐 Quick Tunnel Example

This example demonstrates Cloudflare Quick Tunnels with the Vite plugin.

No hostname or API token required - just run npm run dev and get a public URL!

Features:

  • ✅ Zero configuration
  • ✅ No API token needed
  • ✅ Random trycloudflare.com URL
  • ✅ Instant public access
  • ✅ Perfect for demos and testing

⚠️ Note:

Quick tunnels are temporary and the URL changes each time you restart. For persistent URLs with custom domains, use named tunnel mode instead.
